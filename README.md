# jaehho.github.io

Personal portfolio and CV repository for Jaeho Cho.

## CV & Resumes

The `cv/` directory contains a full CV and targeted one-page resumes, all written in [Typst](https://typst.app/).

| Document | Source |
|---|---|
| Full CV | `cv/cv.typ` |
| Research Resume | `cv/resume-research.typ` |
| SWE Resume | `cv/resume-swe.typ` |

### Compile locally

```bash
typst compile cv/cv.typ
typst compile cv/resume-research.typ
typst compile cv/resume-swe.typ
```

PDFs are generated by CI on every push to `main` and uploaded as workflow artifacts.
